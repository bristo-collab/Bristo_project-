# HealthKart E-commerce Website

A modern, high-performance e-commerce platform for health and nutrition products. This project features a robust frontend, administrative dashboard, and backend services to provide a seamless shopping experience.

## Features

- **Premium UI/UX**: Clean, responsive design with glassmorphism elements and smooth micro-animations.
- **Admin Dashboard**: Comprehensive analytics, real-time order tracking, and inventory management.
- **Product Management**: Detailed product listings with nutrition facts, categories, and reviews.
- **User Experience**: Secure checkout, order history, and profile management.

## Project Structure

- `src/`: Frontend React components, pages, and styles.
- `server/`: Backend API and server-side logic.
- `public/`: Static assets and public files.
- `assets/`: Image assets and design resources.

## Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
